Apa itu Python?

Python adalah bahasa pemrograman yang simple banget, cocok buat kamu yang baru mulai belajar coding. Sintaksnya mirip bahasa manusia, jadi lebih enak dibaca & ditulis.

Contoh Kode Dasar Python

# Ini adalah program Python pertama kamu
print("Halo, dunia!")
name = input("Siapa nama kamu? ")
print("Senang kenal denganmu,", name)

Yuk coba langsung di editor interaktif di bawah ini!

Latihan Seru!

Soal: Buat program Python yang menanyakan nama kamu dan umur kamu, lalu tampilkan kalimat seperti:
Halo, aku X dan umurku 20 tahun.

Cobain di editor di atas ya!
