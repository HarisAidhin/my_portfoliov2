Sejarah Singkat C++

C++ dibuat oleh Bjarne Stroustrup pada awal 1980-an sebagai pengembangan dari bahasa C. Tujuannya, bikin bahasa yang powerful, mendukung pemrograman berorientasi objek, tapi tetap cepat dan fleksibel. Sampai sekarang, C++ masih jadi andalan buat aplikasi performa tinggi, game, sistem operasi, dan lain-lain.

Apa itu C++?

C++ adalah bahasa pemrograman tingkat menengah yang mendukung pemrograman prosedural, berorientasi objek, dan generik. Bahasa ini banyak dipakai untuk membuat software, game, driver perangkat keras, dan aplikasi yang butuh performa tinggi.

Contoh Kode Dasar C++

#include <iostream>
using namespace std;

int main() {
    cout << "Halo, dunia C++!" << endl;
    string name;
    cout << "Siapa nama kamu? ";
    cin >> name;
    cout << "Senang kenal denganmu, " << name << "!" << endl;
    return 0;
}

Kamu bisa coba kode ini langsung di editor online di bawah ini!
