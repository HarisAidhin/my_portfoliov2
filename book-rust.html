Sejarah Singkat Rust

Rust dikembangkan oleh Mozilla Research dan diumumkan pada tahun 2010. Rust dirancang untuk memberikan keamanan memori tanpa garbage collector, sehingga sangat cocok untuk aplikasi sistem dan performa tinggi. Rust terkenal dengan konsep ownership dan borrowing yang membuat kode aman dan efisien.

Contoh Kode Dasar Rust

// Program Rust sederhana
fn main() {
    println!("Halo, Dunia!");
}

Coba jalankan contoh sederhana ini di editor interaktif di bawah!

Latihan Seru!

Buat program Rust yang menanyakan nama dan umur, lalu tampilkan kalimat:

Halo, saya [nama] dan umur saya [umur] tahun.

Coba coding langsung di editor di atas ya!

Tips Belajar Rust

  • Pahami konsep ownership dan borrowing sejak awal.
  • Gunakan println! untuk output ke layar.
  • Eksplor match untuk kontrol alur yang powerful.
  • Jangan takut debugging, compiler Rust sangat membantu.
