Sejarah Singkat Kotlin

Kotlin dibuat oleh JetBrains dan diumumkan pada tahun 2011. Bahasa ini dirancang untuk interoperabilitas penuh dengan Java, sintaks yang lebih bersih, dan mendukung pemrograman fungsional serta berorientasi objek. Kotlin resmi jadi bahasa utama Android sejak 2017.

Contoh Kode Dasar Kotlin

// Program Kotlin sederhana
fun main() {
    println("Halo, Dunia!")
}

Maaf guys kode Kotlin ini perlu dijalankan di aplikasi/situs resmi seperti Kotlin Playground untuk praktek langsung.
