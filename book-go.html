Sejarah Singkat Go

Go, atau Golang, dibuat oleh Google pada tahun 2009 oleh Robert Griesemer, Rob Pike, dan Ken Thompson. Go dirancang untuk memudahkan pengembangan perangkat lunak berskala besar dengan performa tinggi dan syntax yang sederhana. Bahasa ini populer untuk membuat aplikasi backend, cloud, dan sistem yang membutuhkan concurrency.

Contoh Kode Dasar Go

// Program Go sederhana
package main

import "fmt"

func main() {
    fmt.Println("Halo, Dunia!")
}

Contoh ini akan menampilkan tulisan Halo, Dunia! di layar. Yuk coba jalankan langsung di editor interaktif di bawah!

Latihan Seru!

Buat program Go yang menanyakan nama dan umur, lalu tampilkan kalimat:

Halo, saya [nama] dan umur saya [umur] tahun.

Coba coding langsung di editor di atas ya!

Tips Belajar Go

  • Pahami package main dan fungsi main() sebagai pintu masuk program.
  • Gunakan fmt.Println() untuk menampilkan output.
  • Manfaatkan fitur goroutine dan channel untuk concurrency.
  • Jangan takut coba dan eksplor fitur bawaan Go yang powerful.
