Sejarah Singkat Java

Java dikembangkan oleh James Gosling dan timnya di Sun Microsystems pada tahun 1995. Java dirancang untuk menjadi bahasa pemrograman yang bisa berjalan di berbagai perangkat dengan prinsip "Write Once, Run Anywhere". Jadi, kode Java bisa berjalan di Windows, Mac, Linux, bahkan perangkat mobile tanpa perlu diubah-ubah.

Contoh Kode Dasar Java

// Program Java sederhana
public class HelloWorld {
  public static void main(String[] args) {
    System.out.println("Halo, Dunia!");
  }
}

Contoh di atas menampilkan tulisan Halo, Dunia! ke layar. Yuk coba mainin langsung di editor interaktif di bawah ini!

Latihan Seru!

Buat program Java yang menampilkan namamu dan umurmu, lalu tampilkan kalimat:

Halo, saya [nama] dan umur saya [umur] tahun.

Coba coding langsung di editor di atas ya!

Tips Belajar Java

  • Biasakan menulis public static void main(String[] args) sebagai titik awal program Java.
  • Pahami konsep class dan object karena Java adalah bahasa pemrograman berorientasi objek (OOP).
  • Gunakan System.out.println() untuk menampilkan output.
  • Jangan takut coba dan bereksperimen, error itu bagian dari proses belajar.
