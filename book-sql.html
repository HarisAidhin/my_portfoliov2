Sejarah Singkat SQL

SQL (Structured Query Language) dikembangkan oleh Donald D. Chamberlin dan Raymond F. Boyce pada awal 1970-an di IBM. SQL dirancang untuk mengelola data dalam sistem manajemen basis data relasional. Bahasa ini telah menjadi standar industri sejak diperkenalkan oleh ANSI pada tahun 1986.

Kenapa Belajar SQL?

  • SQL digunakan oleh hampir semua sistem database modern seperti MySQL, PostgreSQL, SQL Server, dan Oracle.
  • SQL adalah dasar dari pekerjaan data analyst, backend developer, hingga sysadmin.
  • Bahasa yang powerful namun mudah dipelajari untuk pemula.
  • Memungkinkan kita mengelola dan mengambil data secara efisien.

Contoh Kode SQL

-- Membuat tabel pengguna
CREATE TABLE users (
  id INT PRIMARY KEY,
  nama VARCHAR(100),
  email VARCHAR(100)
);

-- Menambahkan data
INSERT INTO users (id, nama, email)
VALUES (1, 'Rama', 'rama@email.com');

-- Mengambil semua data
SELECT * FROM users;

Tips Belajar SQL

  • Gunakan database seperti SQLite atau MySQL untuk praktik langsung.
  • Pahami perintah dasar: SELECT, INSERT, UPDATE, DELETE, JOIN.
  • Cobalah query latihan di situs seperti SQLZoo, LeetCode, atau Mode SQL.
  • Biasakan menggunakan alias dan filter WHERE untuk efisiensi data.
