Sejarah Singkat PHP

PHP dibuat pertama kali oleh Rasmus Lerdorf pada tahun 1994. Awalnya, PHP itu singkatan dari "Personal Home Page" karena dipakai buat bikin halaman pribadi. Tapi sekarang, PHP sudah jadi singkatan rekursif: "PHP: Hypertext Preprocessor". Gak cuma buat halaman statis, tapi juga website interaktif dan aplikasi web besar.

Apa itu PHP?

PHP adalah bahasa scripting yang jalan di server, sering banget dipakai buat bikin website dinamis. Jadi, PHP itu yang bikin website bisa interaktif dan bukan cuma tampilan statis doang.

Contoh Kode Dasar PHP

<?php
  // Ini contoh script PHP yang nampilkan teks
  echo "Halo, dunia PHP!";
?>

Coba lihat langsung hasilnya di editor interaktif di bawah ini!

Latihan Seru!

Soal: Buat script PHP yang menyapa user dengan nama mereka, contoh output:
Halo, nama saya x!

Gunakan echo dan variabel sederhana.
Cobain di editor di atas ya!

Tips Belajar PHP üéØ

  • Ingat, PHP itu berjalan di server, jadi output-nya bisa kamu lihat di browser setelah di-*render*.
  • Mulai dari kode simpel dulu, jangan langsung kompleks.
  • Pahami cara kerja variabel dan echo buat nampilin data.
  • Jangan takut coba-coba, coding itu belajar dari kesalahan.
